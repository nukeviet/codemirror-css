# codemirror-css
The CSS editor used for NukeViet, built from CodeMirror 6.

**Requirements:**

- node v18+
- npm 10.5+

**Instructions:**

```
git clone https://github.com/nukeviet/codemirror-css.git
cd codemirror-css
npm i
npm run build
```

Then, copy the files from `dist/*` into your project and use them.
